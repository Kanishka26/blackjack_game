# Blackjack Game

A simple browser-based Blackjack game built with plain HTML, CSS, and JavaScript. The goal is to reach a total of 21 without going over.

## How To Run

1. Open `index.html` in your browser.
2. Click **Start Game** to deal the first two cards.
3. Click **New Card** to draw another card, if you are still in the round.

## Game Rules

- The game starts by dealing two random cards.
- Each card is a random number from 1 to 13.
- The card values are added directly to your total.
- If your total is 21, you win with Blackjack.
- If your total goes above 21, you lose.

## Features

- Clean one-page interface
- Random card dealing
- Current hand and total display
- Win and lose messaging

## Project Files

- `index.html` - page structure and game controls
- `style.css` - visual styling
- `script.js` - game logic

## Notes

This version is intentionally minimal and does not include a dealer, betting, card suits, or special face-card/ace rules.
